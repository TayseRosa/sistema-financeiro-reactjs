# 📥 Como usar
```js

    //Clonar o repositório
    $ git clone https://github.com/TayseRosa/sistema-financeiro-reactjs.git

    //Entrar no diretório
    $ cd sistema-financeiro-reactjs

    //Instalar bilbiotecas
    $ npm install

    //Startar o projeto
    $ npm start

```
